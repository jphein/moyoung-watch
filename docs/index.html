Install

You need Home Assistant with at least one ESPHome Bluetooth proxy in range of the watch.

1. Integration (HACS) — add this repo as a custom repository (category Integration), install MoYoung Watch, restart HA, then Settings → Devices & Services → Add Integration → MoYoung Watch.

2. Automations — drop the packages/moyoung_*.yaml you want into HA's packages/ directory.

3. Faces — pre-built .bin faces ship in watchfaces/; rebuild them with:

cd watchfaces/dawft && ./get-dawft.sh     # fetch the GPL packer (once)
cd ../solar-soc && python3 build_tou.py   # → solar-offpeak/partial/peak.bin

4. CLI (optional) — cd client && pip install -e ., then moyoung scan.

The protocol, briefly

The MoYoung-v2 / Da Fit watches speak a small byte protocol on a single GATT service — not the protobuf of the unrelated Actions/Zephyr watches.

PieceDetail
Service0xFEEA — control, data (image chunks), notify & steps characteristics
FramingFE EA | flags | len | cmd | payload…, written to the control characteristic
Facesa MoYoung .bin container of RLE / raw RGB565 blobs + a table of positioned fields
Injectionthe firmware repositions value-driven fields at render time; WEATHER_TEMP takes an arbitrary number
Sourcecontrol/sync ported clean-room from Gadgetbridge's MoYoung coordinator

Full notes in PROTOCOL.md; every embeddable face field is documented interactively in the face builder.
